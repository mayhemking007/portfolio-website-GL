## 📖 About This Portfolio

This portfolio is designed to look and behave like a **GitHub repository**.
Each file in the repository represents a section of my professional profile.

Instead of navigating a traditional website, you can explore my background, skills, and projects by **browsing the repository structure using the sidebar**.

---

## 📂 Repository Navigation

Use the sidebar to explore the portfolio.


| File / Folder   | Description                           |
| --------------- | ------------------------------------- |
| `README.md`     | Overview of this portfolio            |
| `ABOUT.md`      | Background, interests, and philosophy |
| `SKILLS.md`     | Technical stack and tools             |
| `EXPERIENCE.md` | Work experience and engineering focus |
| `projects/`     | Backend and system design projects    |
| `CONTACT.md`    | Contact information and resume        |

Inside the **`projects/`** folder you will find detailed explanations of backend systems I have built.

## 🍴 Fork This Portfolio

If you like the **repository-style portfolio idea**, you can easily fork this project.

Use the **Fork button in the header of this website**.

When you click it:

1. A modal will appear showing the **GitHub repository URL**
2. You can **copy the repository link**
3. Or click **View Source** to open the GitHub repository page

Once on GitHub, you can fork it normally and create your own version.

Example clone command:

```
git clone https://github.com/mayhemking007/portfolio-website-GL.git
```

Steps after forking:

1. Clone your fork locally
2. Update the Markdown files with your own information
3. Deploy your portfolio
