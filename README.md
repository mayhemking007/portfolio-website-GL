# GitHub-Style Developer Portfolio

A **backend-focused portfolio website designed to look and behave like a GitHub repository**.

Instead of navigating a traditional portfolio page, visitors explore the site through **Markdown files, a repository file tree, and project documentation**, similar to browsing a real GitHub repo.

---

## 📂 Repository Navigation

The portfolio is structured like a repository where each file represents a section of the profile.

| File / Folder   | Description                                    |
| --------------- | ---------------------------------------------- |
| `README.md`     | Overview of the portfolio                      |
| `ABOUT.md`      | Personal background and engineering philosophy |
| `SKILLS.md`     | Technology stack and tools                     |
| `EXPERIENCE.md` | Professional experience                        |
| `projects/`     | Detailed documentation of engineering projects |
| `CONTACT.md`    | Contact information                            |

Inside the **`projects/`** folder you will find technical write-ups of systems I have built.

## 🍴 Fork This Portfolio

If you like this repository-style portfolio idea, you are welcome to **fork it and build your own version**.

Steps:

1. Click **Fork** on GitHub
2. Clone your fork

```bash
git clone https://github.com/your-username/portfolio
```

3. Replace the Markdown content with your own information
4. Deploy your portfolio
